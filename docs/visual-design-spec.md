# 食堂仿真 · Minecraft 体素风格视觉设计规范

## 1. 整体风格定位

**核心关键词：** Minecraft 体素 × 低多边形 × 温暖明快

| 维度 | 描述 |
|------|------|
| 构成方式 | 所有元素由方块/长方体组成，无曲面 |
| 纹理 | 纯色为主，不贴图；通过颜色差异表达材质 |
| 色调 | 温暖自然系，避免荧光色 |
| 光照 | 方向光 (45°) + 柔和环境光，投射阴影 |
| 相机 | 透视投影，45° 俯视角，可旋转/缩放 |

### 光照参数建议

| 参数 | 值 |
|------|-----|
| 方向光颜色 | `#FFFFFF` |
| 方向光强度 | `1.2` |
| 方向光位置 | `(15, 20, 10)` |
| 环境光颜色 | `#F5F0E8` (暖白) |
| 环境光强度 | `0.6` |
| 阴影 | 开启，`mapSize: 2048` |

## 3. Agent 模型规格

### 3.1 身体比例

基于 Minecraft 角色的 Q 版变形（2.5 头身比）：

```
         ┌───┐
         │头 │  0.35 × 0.35 × 0.35
         └───┘
        ┌─────┐
        │ 身体 │  0.35 × 0.45 × 0.2
        └─────┘
        ┌┐  ┌┐
        │├──┤│  腿: 0.12 × 0.35 × 0.12 × 2
        └┘  └┘
```

| 部位 | 尺寸 (W × H × D) | Y 偏移 (相对脚底) |
|------|------------------|------------------|
| 头 | 0.35 × 0.35 × 0.35 | 1.15 → 1.5 |
| 身体 | 0.35 × 0.45 × 0.2 | 0.7 → 1.15 |
| 左腿 | 0.12 × 0.35 × 0.12 | 0.0 → 0.35（+ x偏移 -0.1） |
| 右腿 | 0.12 × 0.35 × 0.12 | 0.0 → 0.35（+ x偏移 +0.1） |

**总高度：** ~1.5 场景单位

### 3.2 行走动画

| 参数 | 值 | 说明 |
|------|-----|------|
| 腿部摆动角度 | ±25° (0.44 rad) | 前后摆动 |
| 摆动频率 | `speed * 3` rad/s | 速度越快摆动越快 |
| 插值方式 | `sin(time * freq)` | 正弦平滑 |

### 3.3 状态视觉反馈（未来增强）

| 反馈方式 | 优先级 | 说明 |
|---------|--------|------|
| 身体颜色 | P0 | 当前已实现，核心识别手段 |
| 头顶图标 | P1 | Sprite —— 如 🍽️ 点餐中、⏳ 排队中 |
| 表情气泡 | P2 | 不耐烦时冒 💢，就餐时 😊 |
| 缩放脉冲 | P3 | 状态切换瞬间微缩放 (scale 0.9→1.0) |

## 4. 场景元素设计

### 4.1 餐饮窗口

```
  ┌──────────┐    ← 招牌 (1.2 × 0.4 × 0.1)，各窗口颜色不同
  └──────────┘
  ┌──────────┐    ← 柜台 (1.5 × 1.0 × 1.0)，深红木色
  │          │
  └──────────┘
```

- 柜台用 `#8B4513` 深红木
- 招牌颜色按窗口编号对应色表
- 可选增强：柜台前加菜品方块（小彩色方块表示菜品）

### 4.2 桌椅

**方桌设计：**

```
  ┌──────────┐   ← 桌面 (0.8 × 0.05 × 0.8)
  ├┐        ┌┤
  ││        ││   ← 4 条桌腿 (0.08 × 0.4 × 0.08)
  └┘        └┘
```

- 桌面：`#C4A882` 橡木色
- 桌腿：`#6B4423` 深木
- 占用时桌面变为 `#A5D6A7` 浅绿（有人在吃）

### 4.3 墙壁

- 基色：`#A89888` 暖灰
- 高度：2 场景单位
- 入口/出口：留 5 格间隙
- 未来增强：窗户（墙上挖洞，透明材质）

### 4.4 地面

- 棋盘格：`#E8E0D0` / `#D4C8B0`
- 可选：排队区用浅色线标识走道

## 5. UI 叠加层风格

| 元素 | 风格 | 位置 |
|------|------|------|
| 统计面板 | 半透明深色底 + 白字 | 左下角 |
| 状态图例 | 色块 + 标签 | 右下角 |

## 6. 实现优先级

| 优先级 | 改进项 | 视觉冲击 | 实现难度 | 说明 |
|--------|--------|---------|---------|------|
| **P0** | 更新 STATE_COLORS | ★★★ | ★ | 改色值即可 |

| **P1** | 光照优化 | ★★★ | ★ | 调方向光+环境光参数 |
| **P1** | Agent 腿部行走动画 | ★★★ | ★★ | sin 函数驱动旋转 |
| **P2** | 桌椅四脚细化 | ★★ | ★★ | 4个小方块替换1个中方块 |
| **P2** | 入口/出口标识色 | ★★ | ★ | 门口地面染色 |
| **P3** | 头顶状态图标 | ★★ | ★★★ | Sprite Billboard |
| **P3** | 阴影微调 | ★ | ★★ | shadow map + bias |
